仿网易云音乐的vue2项目，利用rem布局+element-ui+mint-ui适配PC+移动端

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

详见package.json文件
